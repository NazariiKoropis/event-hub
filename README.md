# Event Hub

Вебзастосунок для пошуку подій, купівлі квитків, керування подіями організатора та адміністрування заявок.

## Команди

- `npm run dev` - запуск локального сервера розробки
- `npm run build` - збірка проєкту
- `npm run lint` - перевірка коду ESLint
- `npm run preview` - перегляд зібраної версії
